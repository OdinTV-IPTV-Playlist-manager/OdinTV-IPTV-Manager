# Языковая система IPTV Менеджера

## Структура папки `lang/`

Папка `lang/` содержит языковые файлы для локализации интерфейса.

### Файлы языков

Имя файла должно соответствовать коду языка:
- `ru.php` - Русский язык (🇷🇺)
- `en.php` - English (Английский, 🇬🇧)
- `et.php` - Eesti (Эстонский, 🇪🇪)
- `lv.php` - Latviešu (Латышский, 🇱🇻)
- `lt.php` - Lietuvių (Литовский, 🇱🇹)
- `it.php` - Italiano (Итальянский, 🇮🇹)
- `es.php` - Español (Испанский, 🇪🇸)
- `cs.php` - Čeština (Чешский, 🇨🇿)
- `hy.php` - Հայերեն (Армянский, 🇦🇲)
- `ka.php` - ქართული (Грузинский, 🇬🇪)
- `be.php` - Беларуская (Белорусский, 🇧🇾)
- `ad.php` - Адыгейский (🇷🇺)
- `ba.php` - Башкирский (🇷🇺)
- `ce.php` - Чеченский (🇷🇺)
- `uk.php` - Українська (Украинский)
- `de.php` - Deutsch (Немецкий)
- и т.д.

Формат имени: `{код_языка}.php`

## Добавление нового языка

1. Создайте файл с именем `{код_языка}.php` в папке `lang/`
2. Скопируйте структуру из шаблона выше
3. Переведите все значения на нужный язык
4. Сохраните файл в кодировке UTF-8 без BOM

### Пример для украинского языка (uk.php):

```php
<?php
// lang/uk.php - Українська мова
return [
    'title' => 'IPTV Менеджер плейлистів',
    'current_playlist' => 'Поточний плейлист:',
    'upload_new' => 'Завантажити новий плейлист',
    // ... остальные переводы
];
```

## Как это работает

1. Система автоматически сканирует папку `lang/` на наличие файлов `*.php`
2. Каждый файл загружается и добавляется в меню выбора языка
3. Выбранный язык сохраняется в сессии пользователя
4. При переключении языка страница перезагружается с новым языком

## Примечания

- Все файлы должны быть в кодировке UTF-8
- Ключи массива должны совпадать с оригинальным русским файлом
- Значения `%s` используются для подстановки динамических данных
- Пустые значения будут показывать ключ вместо перевода
